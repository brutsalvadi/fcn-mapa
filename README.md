# FCN Màster 2025-2026 – Mapa interactiu

Mapa interactiu amb tots els events del **Circuit Català de Trofeus de Natació Màster** i els **Campionats de Catalunya Màster** de la temporada 2025-2026, organitzats per la [Federació Catalana de Natació](https://www.natacio.cat).

## Funcionalitats

- 📍 **Mapa** amb un marcador per cada competició (circuit en blau, campionats en vermell)
- 🖱️ **Passa el cursor** sobre un marcador per veure el nom i la data
- 👆 **Clica** sobre un marcador per obrir el panell complet amb adreça, piscina i programa de proves
- 🔘 **Filtre** per amagar els events passats (actiu per defecte)
- 📱 **Responsive** — funciona al mòbil i es pot instal·lar com a app (PWA)

## Instal·lació com a app (iOS / Android)

**iOS:** Obre l'enllaç amb Safari → botó Compartir → *Afegir a la pantalla d'inici*

**Android:** Obre l'enllaç amb Chrome → menú (⋮) → *Afegir a la pantalla d'inici*

## ⚠️ Avís sobre el programa de proves

El programa de proves que apareix a cada event és **orientatiu** i s'ha obtingut a partir de les cròniques i resultats de les edicions anteriors dels mateixos trofeus. No prové de les convocatòries oficials de la temporada 2025-2026.

Pot haver-hi diferències respecte al programa real. Consulta sempre la convocatòria oficial a [natacio.cat](https://www.natacio.cat/natacio-masters-temp-25-26/) per a la informació definitiva.

Els camps marcats amb *"programa parcial"* o *"basat en edició anterior"* indiquen que les dades són especialment incertes.
